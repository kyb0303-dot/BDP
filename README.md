# BDP
## 지하철 승하차 데이터 분석
1. 프로젝트 개요
   
   주제 : 공공데이터를 활용하여 지하철 승하차 인원 분석
   
   목표 :
   1. 연도별 이용객이 많은 역 top 10
   2. 연도별 호선별 이용객이 가장 많은 역
   3. 연도별 호선별 이용객 변화
   4. 특정호선(4호선)의 주요 이용 역
   
   데이터셋 : 공공데이터포털(서울 열린데이터 광장) '서울특별시 지하철 호선별 역별 승하차 인원 데이터’
   (https://www.data.go.kr/data/15071921/fileData.do#)

   
2. 기술 스택
   - HDFS : 원본 데이터 저장
   - Apache Spark : 데이터 전처리
   - Spark SQL : 데이터 분석
   - Python : 결과 데이터 처리
   - Matplotlib : 데이터 시각화

3. 구현계획

   > 데이터 수집
      - 공공데이터포털에서 지하철 승하차 데이터 수집
      - HDFS에 데이터 저장
  
   > 데이터 전처리
      - 사용일자에서 연(year), 월(month), 일(day)분리
      - 승차 승객수와 하차 승객수를 통하여 total_passenger 생성
      - 데이터 타입 변환
      - 결측치 제거
      - 분석에 불필요한 데이터 제거(등록일자)

   > 데이터 분석
      - Spark SQL을 활용하여 다음 4가지 주제 분석 수행
           1. 연도별 이용객이 가장 많은 역 Top 10
           2. 연도별·호선별 이용객이 가장 많은 역
           3. 연도별 호선 이용객 변화
           4. 2026년 4월 4호선 역별 일평균 이용객 순위
              
   > 결과
      - 분석 결과 CSV 형태로 저장
      - Python(Matplotlib)을 통하여 시각화

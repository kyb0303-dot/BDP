# BDP
## 지하철 승하차 데이터 분석
1. 프로젝트 개요
   
   주제 : 공공데이터를 활용하여 지하철 승하차 인원 분석
   
   목표 :
   1. 연도별 이용객이 많은 역 top 10
   2. 연도별 호선별 이용객이 가장 많은 역
   3. 연도별 호선별 이용객 변화
   4. 특정호선(4호선)의 주요 이용 역
   
   데이터셋 : 공공데이터포털 '서울시 시간 평균 대기오염도 정보'
   
3. 기술 스택
   
   Apache Spark/Pandas : 데이터 처리
   
   Linear Regression : 이전 시간대의 대기오염 수치를 기반으로 미래 오염 농도를 예측
   
   PCA : 다양한 오염 지표들을 차원축소하여 대기질 오염의 핵심 성분을 추출
   
   K-means Clustering : PCA를 거친 특징 데이터를 활용하여 대기질 오염의 패턴이 유사한 지역을 그룹화

5. 구현계획

   데이터 수집 : 공공데이터포털에서 특정일의 대기오염 데이터를 수집
   
   데이터 전처리 : 결측치 처리, 데이터 타입 변환, 시간대별/자치구별 처리
   
   탐색적 데이터 분석 : 시간대별 오염도 변화를 시각화, 물질간의 상관계수산출
   
   모델링 및 평가 : PCA를 적용하여 데이터의 군집도 확인, 오염도 예측 수행
